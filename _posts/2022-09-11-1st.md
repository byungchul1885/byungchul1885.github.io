## 설치
```
$ pip install Flask
```

## minimal Flask application
```python
from flask import Flask
app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'
```

## 환경변수 설정
```
# Command Prompt
C:\path\to\app>set FLASK_APP=hello.py

# PowerShell
PS C:\path\to\app> $env:FLASK_APP = "hello.py"
```

## 실행
```
$ flask run
```
