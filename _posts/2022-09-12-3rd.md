## variable length arguments

```python
def adder(*num):
    sum = 0
    
    for n in num:
        sum = sum + n

    print("Sum:",sum)

adder(3,5)
adder(4,5,6,7)
adder(1,2,3,5,6)
```


## variable keyword arguments

```python
def intro(**data):
    print("Data type of argument:",type(data))

    for key, value in data.items():
        print(f"{key} is {value}")

intro(Firstname="Sita", 
    Lastname="Sharma", 
    Age=22, 
    Phone=1234567890)
```

### 결과

```
Data type of argument: <class 'dict'>
Firstname is Sita
Lastname is Sharma
Age is 22
Phone is 1234567890
```
