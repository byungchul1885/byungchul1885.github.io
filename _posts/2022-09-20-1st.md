## 타입 확인 방법 -- 두 가지

```python
>>> variable = "hello"
>>> type(variable) is str
True
>>> isinstance(variable, str)
True
```

## isinstance()을 선호하는 이유

+ isinstance()이 더 빠르다
+ 상속을 포함하여 타입 체킹을 한다

## 상속을 고려한 type 체킹 예제

### 클래스 정의

```python
class MyAwesomeList(list):
    # Add additional functions here
```

### 결과

```python
>>> my_list = MyAwesomeList()
>>> type(my_list) is list
False
>>> isinstance(my_list, list)
True
```
