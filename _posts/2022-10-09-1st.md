```python
lat = list(data["LAT"])
lon = list(data["LON"])
elev = list(data["ELEV"])
```


```python
import folium

map = folium.Map(location=[38.58, -99.09], 
                 zoom_start=5, 
                 tiles="Stamen Terrain")
```


```python
def color_producer(elev):
    if elev < 1000:
        return 'green'
    elif 1000 <= elev < 3000:
        return 'orange'
    else:
        return 'red'    
```


```python
for lt, ln, el in zip(lat, lon, elev):
    map.add_child(
        folium.CircleMarker(location=[lt, ln], 
                            fill_opacity = 0.7,
                            radius = 6,
                            color = 'grey',
                            popup=[str(el) + ' m'], 
                            fill_color = color_producer(el)))    
map
```
