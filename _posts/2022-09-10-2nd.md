## 데코레이터란?
+ 함수를 받아 명령을 추가한 뒤 이를 다시 함수의 형태로 반환하는 함수이다
+ 함수의 내부를 수정하지 않고 기능에 변화를 주고 싶을 때 사용한다
+ 함수의 전처리나 후처리 필요가 있을 때 사용 한다
+ 반복을 줄이고 메소드나 함수의 책임을 확장한다

## 예제
```python
def trace(func):
    def wrapper():
        print(func.__name__, '함수 시작')
        func()
        print(func.__name__, '함수 끝')
    return wrapper
 
@trace
def hello():
    print('hello')
 
@trace
def world():
    print('world')
 
hello()
world()
```

## 실행 결과
```
hello 함수 시작
hello
hello 함수 끝
world 함수 시작
world
world 함수 끝
```

## 데코레이터 여러 개
```python
def decorator1(func):
    def wrapper():
        print('decorator1')
        func()
    return wrapper
 
def decorator2(func):
    def wrapper():
        print('decorator2')
        func()
    return wrapper
 
# 데코레이터를 여러 개 지정
@decorator1
@decorator2
def hello():
    print('hello')
 
hello()
```

## 실행 결과
```
decorator1
decorator2
hello
```

@을 사용하지 않았을 때는 다음 코드와 동작이 같다.

```python
decorated_hello = decorator1(decorator2(hello))
decorated_hello()
```
